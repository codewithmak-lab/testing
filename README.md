# 🎓 University Website

A modern and responsive **University Website** built using **HTML** and **CSS**.  
This project showcases a clean user interface for an educational institution website with sections like Home, About, Courses, Contact, and more.

---

## 📌 Features

- ✅ Responsive Design
- ✅ Clean and Modern UI
- ✅ Navigation Bar
- ✅ Hero Section
- ✅ About University Section
- ✅ Courses / Departments Section
- ✅ Contact Form
- ✅ Footer Section
- ✅ Pure HTML & CSS Project

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
